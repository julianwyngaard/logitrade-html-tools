# Base64 Encoder / Decoder

Encode text to Base64 or decode Base64 back to text with live conversion.

## Features

- **Encode mode** - Convert plain text to Base64
- **Decode mode** - Convert Base64 back to plain text
- **Live conversion** - Output updates as you type
- **UTF-8 support** - Handles multi-byte characters (emoji, accented characters, etc.)
- **URL sharing** - Share the current input and mode via URL
- **localStorage** - Persists your last input and mode between sessions

## Usage

1. Select **Encode** or **Decode** mode using the toggle at the top
2. Type or paste content into the input pane
3. The output updates automatically as you type
4. Use **Copy** to copy the output, or **Share URL** to generate a shareable link
